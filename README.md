Project Overview
This project demonstrates SQL data cleaning and exploratory data analysis (EDA) using a real-world layoffs dataset.
Skills Used
•	SQL 
•	MySQL 
•	Window Functions 
•	CTEs 
•	Data Cleaning 
•	Aggregate Functions 
•	GROUP BY 
•	ORDER BY 
•	Date Functions 
Cleaning Steps
•	Removed duplicate records 
•	Standardized industry names 
•	Trimmed country values 
•	Converted text dates into DATE format 
•	Filled missing industries 
•	Removed rows with insufficient information 
EDA Questions
•	Which companies laid off the most employees? 
•	Which industries were most affected? 
•	Which countries experienced the highest layoffs? 
•	Layoffs by funding stage 
•	Layoffs over time
